# monitoramento-tensao-led-sensor-optico
Este projeto foi desenvolvido como parte do processo seletivo para ingresso em uma equipe de competição universitária, com foco na avaliação de habilidades práticas em sistemas embarcados e eletrônica aplicada.

O sistema implementa um monitoramento de tensão utilizando LEDs como indicadores visuais, com variação simulada por um potenciômetro, permitindo a análise de leituras analógicas e tomadas de decisão por meio de lógica condicional. Além disso, o projeto inclui o acionamento de LEDs a partir de um sensor óptico, explorando a interação entre sensores e atuadores em tempo real.

O desenvolvimento envolveu conceitos fundamentais como leitura analógica, controle de saídas digitais, prototipagem em protoboard e integração entre hardware e software, sendo documentado por meio de imagens do circuito e registros do código-fonte.
